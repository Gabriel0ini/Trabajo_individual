# Resumen: Git básicoGabriel Jhoan Cespedes Garcia## ¿Qué es Git?Sistema de control de versiones distribuido (VCS) que permite guardar y gestionar cambios en archivos a lo largo del tiempo de forma local.## ¿Cómo nació Git?Git fue creado en 2005 por Linus Torvalds durante el desarrollo del kernel de Linux, cuando la comunidad perdió acceso a la herramienta que utilizaban antes, llamada BitKeeper. Ante la necesidad de seguir coordinando el trabajo de muchos desarrolladores y no depender de software propietario, decidió crear su propio sistema de control de versiones,que lo creó en 2 o 3 semanas que se quedo desarrollando en su casa.Diseñó Git para que fuera rápido, distribuido y seguro, permitiendo que cada desarrollador tuviera una copia completa del proyecto y pudiera trabajar de manera eficiente sin depender de un servidor central.## ¿Cómo instalar Git?- Ir a: https://git-scm.com/install/- Seguir pasos según el sistema operativo Linux o Windows- Verificar instalación con:  git --version## Configuraciones básicas- git config --global user.name "Gabriel Jhoan Cespedes Garcia"- git config --global user.email "gabrijhhoan2006@gmail.com"## Archivos esenciales en un repositorio- README.md → descripción del proyecto- .gitignore → archivos que Git debe ignorar
